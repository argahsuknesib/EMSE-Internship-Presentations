\begin{frame}
    \frametitle{Regarding the log file}
    Itemizing the characteristics of the log file
    \begin{itemize}
        \item A 3 by 3 robot's space grid log file has been created.
        \item The same method can be enhanced for a 5 by 5 or a 7 by 7 grid space.
        \item The log file is updated at a frequency of 100Hz i.e 0.01 seconds which is exactly the 'odom' topic's update frequency.
        \item File is arranged as one-one, one-two \dots two-three \dots three-three.
        \item two-two position is given to the robot.
        \item manhattan and euclidean distances to the robot are calculated and added.
        \item the z-axis's orientation provided in degrees for calculation of the robot's orientation.
    \end{itemize}
\end{frame}
